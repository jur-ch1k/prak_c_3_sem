Доделать задачу с семинара :

1) Даны два файла, содержащие последовательности вещественных чисел (не в текстовом виде),
упорядоченных по возрастанию. Объединить их в один файл, также упорядоченный по
возрастанию. Имена исходных файлов и файла, в который должен быть записан результат,
передаются в командной строке
Для тестирования создать файлы, содержащие последовательности вещественных чисел (не в текстовом виде), для записи использовать fwrite().
Написать программу для вывода такого файла на экран в читаемом виде.

2) Удалить из файла все строки, длина которых больше заданного числа n.
Символ перевода строки в длину не включать. Длину строки, состоящей только из '\n' считать равной 0. В конце последней строки файла '\n' может отсутствовать.
Имя файла и число n передаются в командной строке. Длина строк в файле не ограничена.
