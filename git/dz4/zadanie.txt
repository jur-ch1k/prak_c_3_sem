1. В командной строке передается имя файла. Реализовать быстрый доступ к строкам файла по номеру строки.
В цикле вводятся со стандартного ввода номера строк и выводятся на экран соответствующие строки. 
Ввод 0 — признак конца работы. Минимизировать количество проходов по файлу.


2 . Написать программу, работающую подобно команде head, которая по умолчанию выводит на экран первые 10 строк файла.
Вызов: > head [-n <целое_число>] [-n -<целое_число>]

[-c <целое_число>] [-c -<целое_число>] [-v] <список файлов>

Опции : одновременно могут быть заданы одна из -n или -c, -v сочетается с любой из них. Наличие опций при вызове необязательно, если их нет, происходит работа по умолчанию.
-n <целое число K> - целое число K задает количество первых строк файла, которые должны быть выведены (вместо 10 по умолчанию).
-n -<целое число K> - выводить все строки файла, кроме K последних строк.
-c <целое_число K> - целое число задает, сколько байтов (символов) от начала файла должны быть выведены.
-c -<целое_число K> для вывода всех байтов файла, кроме K последних.
-v — перед выводом печатать название файла.
