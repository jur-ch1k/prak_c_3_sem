Строим бинарное дерево. Тип элемента данных — int.

1) Написать функцию вставки нового элемента в бинарное дерево поиска - рекурсивный вариант

2) Написать функцию вставки нового элемента в бинарное дерево поиска -  вариант без рекурсии

3) Написать функцию для вывода дерева на экран.

4) Написать функцию для удаления дерева.

В main() в цикле вводить элементы и строить два дерева ( функции 1) и 2)). Вывести на печать результат.

5) Написать функцию, которая по дереву строит двунаправленный список.

6) Написать функцию для вывода списка на экран.

7) Написать функцию для удаления из списка элемента с заданным значением.

8) Написать функцию для удаления списка.

В main() построить по дереву список, вывести его на экран, продемонстрировать удаление элемента из середины, первого и последнего элементов.
По окончании работы деревья и список удалить, всю динамически выделенную память освободить.
